# Bookmarklet-Roulette

This is a bookmarklet that randomly changes the attributes on a site. Here's a list of everything it can do:

1. Randomly change the background color
2. Randomly change the text color
3. Make all text bold
4. Make all text italics
5. Make all text as thin as the font allows it to
6. Randomly rotates each element
7. Randomly skews each element
8. Changes the allignment of all elements
9. Removes all padding and margins
10. Deletes all elements
11. Makes all text Comic Sans
12. Randomly changes the font-size of all text
13. Randomly changes the opacity of all text
14. Makes the screen blurry
